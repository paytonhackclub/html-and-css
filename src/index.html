What is HTML?

  • a set of rules on the web
  • a acronym for Hyper Text Markup Language
  • a structured language composed of open-tags and closed tags
  • the back bone of the web (or most at the very least)
  • works with CSS to change the predefined rules.
    <p>I've been made an example of!</p>

What is CSS?

  • a way to change the rules that govern how HTML is displayed.
  • a acronym for Cascading Style Sheets
  • a structured languages
  • the backbone of design on the web
    .example
    {
        color: #ccc;
    }
